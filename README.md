# Portfolio (Light Theme) — Md. Shadman Sakib Rahman

Minimal, corporate-style interactive portfolio with 3D floating elements.

**Live:** [shadmansakibrahman.github.io/portfolio-light](https://shadmansakibrahman.github.io/portfolio-light/)

## Features
- 3D floating translucent orbs (Three.js)
- Clean corporate design with Manrope typography
- Smooth scroll-triggered animations
- Fully responsive (mobile, tablet, desktop, ultrawide)
- Custom cursor with hover effects

## Built With
HTML, CSS, JavaScript, Three.js
